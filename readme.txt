== Description ==
This is Social Media Bar plugin to social media icons on the site right side for all pages.

== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "TP Social Media Bar". Click on Install and then Activate the Plugin.
That's it, "TP Social Media Bar is now activated on your site!

Or

1. Upload `tp-social-media-bar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by going to the TP Social Media Bar menu from settings.
